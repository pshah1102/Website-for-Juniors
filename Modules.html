Abcdgenics

Abcdgenics focuses on kids of age group 4-5 for teaching them how to pronounce alphabets in a gamified manner. A button is provided for alphabet systhesis which speaks alphabets randomly and the kid has to select the correct alphabet. If the selected alphabet is correct the alphabet will turn green in colour else it will turn to red meaning wrong answer.







Wonder Speak

Wonder Speak is a unique module where we try to test the speaking skills of the child. An object along with its image is provided as a question and the child has to speak it. We use Speak Recognition to check whether the child has correctly identified the object's name and also pronounced it correctly or not.
